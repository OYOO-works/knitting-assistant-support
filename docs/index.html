Features

編みアシでできること

📄

編み図を取り込んで表示

手持ちの編み図のPDFや写真をアプリに取り込んで表示。段帯ハイライトが現在の段を常に示します。

🖼️

写真の編み図にも対応

カメラロールの写真として保存した編み図も取り込めます。PDFがなくても安心です。

🎙️

音声操作で段数を進める

「次の段」と話しかけるだけで1段前進。両手が編み物でふさがっていても問題ありません。

🤙

Siriショートカット対応

「Hey Siri、編みアシ次の段」と言えば、他のアプリを使いながらでも段数を進められます。

🔧

段帯のキャリブレーション

段高さ・上下端をステップガイドで正確に設定。どんな編み図にも合わせられます。

🔢

段数カウンターとしても使える

PDFも写真もなくてOK。シンプルな段数カウンターとして棒針・かぎ針どちらにも利用できます。

How to Use

かんたん3ステップ

1

プロジェクトを作成する

「新規作成」をタップして、PDFや写真の編み図をインポート。タイトルを付けて保存します。段数のみの管理も可能です。

2

段帯を編み図に合わせる

「高さ調整」でキャリブレーションを実行。段帯の上端・下端を実際の段に重ねることで、正確な位置追跡ができます。

3

編みながら段を進める

「次の段」ボタン、音声、またはSiriで段を進めます。現在の段番号が大きく表示され、編み図（写真・PDF)上の段帯も自動でスクロールします。

FAQ

よくある質問

アプリについて

Q 編みアシとはどんなアプリですか？

編みアシ（あみあし）は、iPhone向けの編み物アシスタントアプリです。編み図のPDFを読み込み、現在の段をハイライト（段帯）でオーバーレイ表示しながら編み進めることができます。また、Siriなどの音声操作で段数の増減ができるため、手を離さずに段数を数え進めます。Siriなどの音声操作で手を離さずに今の段数を確認することもできます。

Q どのデバイスに対応していますか？

編みアシはiPhoneに対応しています。iOS 16.2以上が必要です。iPadでも操作可能です。

Q 棒針編みとかぎ針編みの両方に使えますか？

はい。編みアシは編み図管理機能付きの段数を管理するアプリのため、棒針編み・かぎ針編みを問わずご利用いただけます。

Q 複数の編み図プロジェクトを管理できますか？

はい。アプリ内で複数のプロジェクトを作成・管理できます。それぞれのプロジェクトに編み図と段数の進捗が保存されます。

使い方

Q 編み図PDFはどのように読み込みますか？

アプリ内のプロジェクト作成画面からPDFファイルを選択して読み込みます。現在は1ページのPDFに対応しています。PDFはiPhone内のファイルアプリやiCloudから選択できます。

Q Siri以外の音声操作はどのように使いますか？

「次の段」などの音声コマンドで段数を操作できます。Siri以外の音声操作を使うには、iPhoneのマイク権限と音声認識権限をアプリに許可する必要があります。権限の許可は、iPhoneの設定アプリ →「プライバシーとセキュリティ」→「マイク」または「音声認識」から行えます。

Q 段帯（ハイライト）がずれます。修正できますか？

「高さ調整」という名前のキャリブレーション機能で調整できます。プロジェクトの画面から「高さ調整」を開き、段の高さ・上端・下端を実際の編み図に合わせて調整してください。編み図によって段の高さが異なるため、最初に一度合わせることで正確に段の帯（ハイライト・オーバーレイ）が表示されます。

権限・設定

Q マイクや音声認識の権限を許可しなくても使えますか？

はい、使えます。音声操作の権限を許可しなくても、PDF表示・段帯のオーバーレイ表示・手動での段数増減など、主要な機能はすべて利用できます。

Q アプリが要求する権限は何ですか？

編みアシが利用する権限は「マイク」と「音声認識」の2つです。どちらも音声操作機能のために使用されます。権限を許可しない場合でも、アプリのPDF表示や手動操作機能は正常に動作します。

Q 1ページのPDFしか対応していないのはなぜですか？

現在のバージョンでは、シンプルさを重視して1ページPDFを前提とした設計になっています。複数ページのPDF対応は今後の機能追加として検討中です。

マイク・音声認識の利用について

本アプリでは音声操作機能のためにマイクおよび音声認識を使用します。

音声データは端末上で処理され、保存・送信は行いません。

データの保存先

プロジェクト・編み図PDF・写真・段数などのデータはすべてお使いのiPhone内にローカル保存されます。外部サーバーへの送信は行いません。
